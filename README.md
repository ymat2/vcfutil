# README

A helper package for VCF manipulation written in python.

## Installation

Initiate a virtual environment and install vcfutil into the venv:

```sh
mkdir your_proj && cd your_proj
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install git+https://github.com/ymat2/vcfutil.git
```

Alternatively,
if [`uv`](https://docs.astral.sh/uv/) is installed,
you can install vcfutil via [`uv tool install`](https://docs.astral.sh/uv/reference/cli/#uv-tool-install):

```sh
uv tool install git+https://github.com/ymat2/vcfutil.git
```
